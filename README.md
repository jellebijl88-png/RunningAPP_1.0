# 🏃 Strive Ultimate Pro+ Hardloop Schema App

Complete Nederlandse hardloop trainingsapplicatie met interval timer, GPS tracking, voortgang bijhouden en persoonlijke schema's.

## ✨ Kenmerken

✅ **5 complete trainingsschema's**:
- Basis Conditie (0-3km)
- Beginner 5KM Plan
- Ervaren Loper 5KM Plan
- 10KM Interval Focus
- Halve Marathon (21.1km)

✅ **Functionaliteiten**:
- Realtime GPS tracking
- Stem begeleiding (Nederlands)
- Interval timer met trilling feedback
- Automatische pauze detectie
- Persoonlijke records bijhouden
- Streak teller
- Geschiedenis met kaart weergave
- Pace grafieken
- Dark / Light thema
- Aanpasbare warming-up / cooling-down
- Eigen persoonlijke schema generator
- 30 minuten baseline test

## 🚀 Gebruik

1. Open `index.html` in een moderne browser
2. Geef toestemming voor locatie
3. Kies je trainingsschema
4. Start je training!

## 📱 **Perfect voor mobiel gebruik (PWA compatible)

## 📦 Installatie voor GitHub Pages

Deze app is 100% frontend, geen backend nodig. Alle data wordt lokaal opgeslagen in de browser.

```bash
git init
git add .
git commit -m "Eerste commit"
git remote add origin <jouw github repo url>
git push
```

Activeer daarna GitHub Pages op de `main` branch in je repository instellingen.

## 🔧 Technisch

- Pure HTML/CSS/JavaScript
- IndexedDB voor permanente opslag
- Leaflet voor kaarten
- Chart.js voor grafieken
- Web Speech API voor stem
- Geolocation API
- Wake Lock API
